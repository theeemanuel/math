# File Descriptions
[CanonicalForm.cpp](https://github.com/theeemanuel/math/blob/main/matrix/CanonicalForm.cpp) performs a simple [matrix diagonalisation](https://en.wikipedia.org/wiki/Diagonalizable_matrix#Diagonalization) to obtain the [Canonical Form of any given Equation](https://en.wikipedia.org/wiki/Canonical_form).
